# Assistente Virtual Extremamente Aesthetic ;)

## 📒 Descrição
Este vídeo é uma breve apresentação de um projeto de assistente virtual.

## 🤖 Tecnologias Utilizadas
- Remini AI para criação da personagem atráves de uma foto minha
- ChatGPT para criação do roteiro
- IA generativa DI-D para geração do avatar virtual 

## 🧐 Processo de Criação
Usei o Remini para criar uma personagem executiva inspirada na minha aparência enviando minha foto para a geração da imagem, Usei o "ChatGPT" para escrever e revisar o texto do vídeo e por fim utilizei O "D-ID" para criar um avatar virtual e voz sintetizada, trazendo uma dimensão futurística ao vídeo.
